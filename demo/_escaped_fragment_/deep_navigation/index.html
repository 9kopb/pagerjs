Deep Navigation

You can navigate into pages in pages, or pages in pages in pages. Just set the href to the same ID as the page.

This is the default sub page. Show the second page. 
  1. <div data-bind="page: {id: 'deep_navigation'}">
  2.     ...
  3.     <div data-bind="page: {id: 'start'}">
  4.         ...
  5.         <a href="#!/deep_navigation/second">Show the second page</a>.
  6.     </div>
  7.     <div data-bind="page: {id: 'second'}">
  8.         ...
  9.         <a href="#!/deep_navigation">Go back to the first sub page</a>.
  10.     </div>
  11.     ...<a href="#!/load_external_content">Go on</a>...
  12. </div>
