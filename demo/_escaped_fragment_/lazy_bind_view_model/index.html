Change Binding Context on Show

When constructing an application with multiple pages, you might want to delay data binding the pages until they are first displayed. Use withOnShow: to asynchronously data bind a page when it is first displayed.

Show page and lazy load view model 

  1. <div data-bind="page: {id: 'start'}">
  2.     <p>
  3.         <a class="btn" data-bind="page-href: '../invention'">Show page and lazy load view model</a>
  4.     </p>
  5. </div>
  6.  
  7. <div class="well" data-bind="page: {id: 'invention', withOnShow: requireVM('invention')}">
  8.     <h3 data-bind="text: name"></h3>
  9.     <p data-bind="text: description"></p>
  10.     <a data-bind="page-href: '../'">Hide this page</a>
  11. </div>
  12.  
  13. <script type="text/javascript>
  14. var requireVM = function (module) {
  15.     return function (callback) {
  16.         require([module], function (mod) {
  17.             callback(mod.getVM());
  18.         });
  19.     };
  20. };
  21. </script>
