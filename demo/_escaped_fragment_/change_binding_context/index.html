Change Binding Context

Use with: to change the binding context - similar to the standard KnockoutJS with binding. 

  1. <div data-bind="page: {id: 'user', 'with': user}">
  2.     <span data-bind="text: name"></span>
  3. </div>
  4. <script type="text/javascript">
  5.     viewModel = {
  6.         user:{
  7.             name:ko.observable("Amy")
  8.         }
  9.     };
  10. </script>
  11.
