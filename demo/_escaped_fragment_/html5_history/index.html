HTML5 History API

The easiest way to use the HTML5 History API is to include History.js. Five lines of code is all it takes after that! 

  1. // use HTML5 history
  2. pager.useHTML5history = true;
  3. // use History instead of history
  4. pager.Href5.history = History;
  5.  
  6. // these two lines are as normal
  7. pager.extendWithPage(viewModel);
  8. ko.applyBindings(viewModel);
  9.  
  10. // start pager.js
  11. pager.startHistoryJs();
