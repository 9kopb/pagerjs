Cached Lazy Loaded Content

Lazy loaded content can be cached. Data binding will still work. The view model's name (Pelle) is data bound in this text. Add sourceCache: true 

  1. <div data-bind="page: {
  2.       id: 'cached_lazy',
  3.       sourceCache: true,
  4.       sourceOnShow: 'cached_lazy.html .content'}">
  5.     ...
  6. </div>
