Configure an iframe

Often you might need to configure the iframe element used. Just include an iframe element inside the page and that iframe will be used. 

  1. <div data-bind="page: {id: 'start', frame: 'iframe', source: 'character/fry.html'}">
  2.     <iframe sandbox="" height="100" width="920"></iframe>
  3. </div>
